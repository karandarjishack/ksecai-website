# KsecAI Website

KsecAI Website is a static marketing and services site for a cybersecurity and AI consulting brand focused on enterprise architecture, SOC engineering, Zero Trust, and AI security.

The project combines a bold landing page with an interactive 3D visual layer and a lightweight front-end scoring experience.

## Highlights

- Service-focused single-page website
- Three-dimensional hero and scene effects
- Cybersecurity service pillars and technology coverage
- Interactive security score widget
- Responsive layout with custom animations and card tilt effects

## Technology Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Three.js-style 3D scene scripting

## Running Locally

No build step is required.

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
